## 1. Summary
- **Delta**: DE-005 — Outline and context generation
- **Specs Impacted**: SPEC-001
- **Desired Outcome**: Pipeline produces sparse outline (JSON + XML) and agent briefing context.md as always-written artifacts, with `--include-hidden` flag for outline visibility control.

## 2. Context & Constraints
- **Current Behaviour**: Pipeline produces normalized JSON and tokens-used but no navigation or briefing artifacts. Normalized JSON not written to disk. Manifest marks outputs as optional.
- **Target Behaviour**: FR-011 (outline), FR-012 (context.md), NF-002 (determinism). All new outputs always written. Manifest fields required.
- **Dependencies**: DE-003 (completed), DE-004 (completed). Role inference is live.
- **Constraints**: ADR-001 zero lint warnings, ADR-002 `mise run` gate. DE-005 boundary: projection/output only, no NormalizedNode changes.
