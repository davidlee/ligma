## 1. Summary
- **Delta**: DE-002 – Project scaffold and Figma client
- **Specs Impacted**: SPEC-001
- **Desired Outcome**: Working CLI that fetches a Figma node subtree + PNG, writes raw JSON + image to artifact directory with manifest. Foundation for all subsequent deltas.

## 2. Context & Constraints
- **Current Behaviour**: No code exists
- **Target Behaviour**: `figma-fetch "<url>" --token "$TOKEN" --out ./artifacts` produces `artifacts/manifest.json`, `artifacts/visual/frame.png`, `artifacts/structure/raw-node.json`
- **Dependencies**: None (first implementation delta)
- **Constraints**: Node 18+ (native fetch), minimal dependencies per NF-006
