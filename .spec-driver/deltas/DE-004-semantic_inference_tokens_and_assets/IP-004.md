## 1. Summary
- **Delta**: DE-004 — Semantic inference, tokens, and assets
- **Specs Impacted**: SPEC-001, PROD-001
- **Design Revision**: [DR-004](./DR-004.md)
- **Desired Outcome**: Every null placeholder DE-003 left behind is populated — role inference, component/variable/asset extraction, semantic flags, text kind, tokens-used summary — via a three-layer architecture (extraction → assembly → inference).

## 2. Context & Constraints
- **Current Behaviour**: `NormalizedNode` carries null for role, component, variables, asset; false for all semantic flags; "unknown" for text.semanticKind. Structurally complete but semantically inert.
- **Target Behaviour**: Per DR-004 — 13-rule role inference, component/variable/asset extractors, text-kind derivation, semantic flag derivation, tokens-used.json summary artifact.
- **Dependencies**: DE-003 complete (core normalization, extractor pattern, NormalizedNode schema)
- **Constraints**: NF-001 2.0x size ceiling, NF-003 pure-function heuristics, ADR-001 zero lint warnings, ADR-002 `mise run` gate
