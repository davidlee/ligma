## 1. Summary
- **Delta**: DE-003 – Core normalization engine
- **Specs Impacted**: SPEC-001
- **Desired Outcome**: `normalize(rawNode)` produces a recursive `NormalizedNode` tree with type classification, bounds, layout, appearance, and text extraction. Wired into `orchestrate()`. Output satisfies RE-002 efficiency metric (fixed field set + ≤2.0x size ceiling).

## 2. Context & Constraints
- **Current Behaviour**: `orchestrate()` returns raw Figma JSON as `unknown`. No structural transformation.
- **Target Behaviour**: FR-005 (normalized model), FR-006 (type classification), FR-008 (layout), FR-009 (text), NF-001 (representation efficiency, RE-002), NF-003 (pure functions)
- **Dependencies**: DE-002 (project scaffold, raw types, Figma client, error hierarchy)
- **Constraints**: All extractors pure functions (NF-003). Zod for schema validation. ADR-001 zero lint warnings. No cross-extractor dependencies.
